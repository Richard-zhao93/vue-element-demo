# vue-element-demo
一个后台前端解决方案，基于 vue 和 element-ui 实现。使用了 vue 技术栈，内置 i18 国际化解决方案，动态路由，权限验证等功能。

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Lints and fixes files
```
npm run lint
```
